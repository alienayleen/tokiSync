# 🚀 TokiSync v1.8.0 정식 릴리스 노트

기존 v1.7.5 이후 기나긴 테스트 페이즈를 마치고, 엔진의 범용화와 소설 뷰어의 완성도를 극대화한 **v1.8.0 정식 릴리스**를 발표합니다.

---

## 🌟 핵심 변경 사항

### 1. ⚙️ 동적 파이프라인: GenericParser 아키텍처 도입
- **하드코딩 탈피**: 사이트별로 작성되던 개별 파서 클래스를 폐지하고, JSON 기반의 동적 규칙 엔진(`RuleManager`)을 도입했습니다.
- **실시간 대응**: 사이트의 HTML 구조가 변경되어도 코드 수정 없이 설정에서 JSON 규칙만 변경하여 즉시 정상화할 수 있습니다.
- **카테고리 확장**: 웹툰, 망가, 소설 등 카테고리별 맞춤형 파싱 로직을 공통 엔진에서 처리합니다.

### 2. 🛡️ 소설 콘텐츠 복호화 엔진 구축
- **보안 돌파**: 최신 안티 스크래핑 기술이 적용된 소설 사이트의 콘텐츠를 API를 통해 안전하게 추출합니다.
- **정밀 복호화**: XOR 및 HMAC 기반 인증 절차를 자동화하여, 이미지나 텍스트가 암호화된 경우에도 완벽한 데이터를 보장합니다.

### 3. 📊 뷰어 2.0: 소설 리더 기능 대폭 강화
- **소설 전용 툴바**: 독서 환경 최적화를 위해 V1의 핵심 기능을 V2로 완벽 이식했습니다.
  - **테마**: Light, Sepia, Dark 모드 지원.
  - **폰트**: 글자 크기(A+/A-) 및 줄 간격 실시간 조절.
  - **레이아웃**: 2쪽 보기(Double Spread) 모드 탑재.
- **🎯 DOM 기반 정밀 동기화**: 폰트 크기나 테마 변경으로 인해 페이지 레이아웃이 변하더라도, 현재 읽고 있던 문단(`logicalIndex`)을 정확히 추적하여 화면 중앙에 유지합니다.

---

## 🛠️ 기술적 개선 사항

- **Webpack 5 & Vite 7**: 빌드 도구를 최신화하여 번들 크기를 줄이고 로딩 속도를 개선했습니다.
- **AbortController 적용**: 뷰어 종료 시 불필요한 네트워크 요청을 즉시 중단하여 리소스 낭비를 방지합니다.
- **IndexedDB L2 캐시**: 다운로드된 에피소드 데이터를 로컬에 영구 저장하여 재방문 시 대기 시간 없이 즉시 열람이 가능합니다.
