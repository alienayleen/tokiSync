# Release Notes v1.8.3 (Stable)

## 📋 개요
이번 v1.8.3 업데이트는 사용자 피드백을 반영하여 **UI/UX를 전면 개편**하고, 소설 다운로드의 **자율성과 정밀도**를 높이는 데 집중했습니다. 특히 기존의 아코디언 메뉴에서 탭 시스템으로 전환되어 더 넓고 쾌적한 조작이 가능해졌습니다.

## 🚀 주요 변경 사항

### 1. UI/UX 전면 현대화
- **520px 확장 레이아웃**: 모달 너비를 확장하여 더 많은 정보를 한눈에 볼 수 있습니다.
- **탭 시스템 도입**: `다운로드`, `설정`, `시스템` 탭으로 기능을 분리하여 접근성을 향상시켰습니다.
- **글래스모피즘 디자인**: 심미적으로 수려한 반투명 배경 효과와 부드러운 애니메이션을 적용했습니다.

### 2. 소설(Novel) 엔진 고도화
- **TXT 포맷 지원**: EPUB 외에도 일반 텍스트 파일로 소설을 저장할 수 있습니다.
- **API 복호화 정제**: 암호화 데이터 수신 시 포함되는 불필요한 메타데이터를 완벽히 제거합니다.
- **속도 제한 해제 (중요)**: 소설 API 복호화 시 강제되던 `very_slow` 대기 시간을 해제했습니다. 이제 사용자가 설정한 `agile`, `cautious` 등의 정책을 그대로 따릅니다.

### 3. 다운로드 안정성 및 리포팅
- **실패 리포트 자동 생성**: 다운로드 누락 발생 시 상세 내역이 담긴 텍스트 파일을 생성합니다.
- **안티-슬립 강화**: 백그라운드 다운로드 중 브라우저 절전 모드로 인한 중단을 방지하는 로직이 개선되었습니다.
- **GAS 설정 유효성 검사**: 설정이 미비할 경우 불필요한 네트워크 통신을 차단하여 안정성을 높였습니다.

## 🛠 수정 및 개선
- 모달 내부 변수 중복 선언으로 인한 빌드 오류 수정
- 설정 모달의 스크롤바 디자인 및 레이아웃 최적화
- `CHANGELOG.md` 및 가이드 문서 현행화
